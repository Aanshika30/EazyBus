# EazyBus
 A smart software solution that automates bus duty scheduling and route planning for Delhi Transport Corporation. Instead of manually creating schedules with spreadsheets, our system uses intelligent algorithms to instantly assign buses and crews while ensuring rest periods and avoiding conflicts.

Key Features

Automated Duty Scheduling
Uses scheduling algorithms to assign buses and crews efficiently while ensuring required rest periods.

Conflict Detection
Prevents scheduling overlaps and operational conflicts in duty assignments.

Interactive Route Planning
Includes a map interface where planners can draw new routes and instantly visualize overlaps with existing routes.

Network Optimization Insights
Helps planners analyze route coverage and improve transport network efficiency.

Technologies Used

Node.js – Backend logic and scheduling services

Flask – API services

MySQL – Database for route and scheduling data
